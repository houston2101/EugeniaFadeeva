Блок навигации

  • Ширина 15 em
  • Должен присутствовать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в примере

Статья 1

Размер блока:
Занимает всю доступную величину
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу равный 2em при помощи margin-bottom

Статья 2

Размер блока:
Занимает всю доступную величину
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу равный 2em при помощи margin-bottom Картинка вставлена при помощи тега, плавание организовано по правому краю внутри статьи.

Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещённым внутрь контентом
